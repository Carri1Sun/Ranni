下面这个版本压缩到 **10 个产物形式模板 + 10 个风格方向**，每个都给你的 agent 一个大概执行指导。

## 10 个产物形式模板

| 模板             | 适合场景                      | agent 执行指导                                                                   |
| -------------- | ------------------------- | ---------------------------------------------------------------------------- |
| 1. 产品介绍页       | SaaS、AI 工具、App、插件、小产品     | 页面目标是让用户快速理解产品价值。结构建议：Hero、核心卖点、功能展示、使用场景、产品截图、用户评价、价格/CTA。重点突出“这个产品解决什么问题”。 |
| 2. 等候名单 / 预告页  | 产品未上线、内测招募、项目预热           | 页面要短、清晰、有期待感。结构建议：大标题、产品一句话介绍、亮点 3 个、上线时间/进度、邮箱收集表单、社交证明。重点是让用户留下联系方式。       |
| 3. 功能发布页       | 新功能上线、版本更新、单个能力介绍         | 聚焦一个功能，不要铺太开。结构建议：功能名称、使用前后对比、核心能力、操作流程、适用人群、立即体验按钮。适合配产品截图或动效演示。            |
| 4. 知识讲解页       | 解释概念、技术科普、课程内容、学习资料       | 页面目标是让用户看懂一个主题。结构建议：主题介绍、背景问题、核心概念、分步骤解释、例子、总结、延伸阅读。适合加入图示、类比、流程图。           |
| 5. 交互式教程页      | 教用户完成某个任务，比如部署网站、使用工具     | 页面要有明确步骤。结构建议：教程目标、准备条件、步骤导航、每一步说明、代码块/截图、常见错误、完成检查。重点是降低用户操作成本。             |
| 6. 个人主页        | 创始人、开发者、设计师、独立创作者         | 页面要建立个人可信度。结构建议：个人简介、当前身份、做过的项目、技能/方向、文章/作品、联系方式。语气可以更个人化，视觉要有记忆点。           |
| 7. 作品集页        | 设计作品、开发项目、摄影、案例展示         | 页面核心是展示成果。结构建议：封面介绍、精选项目网格、单个项目详情、角色贡献、结果数据、联系入口。重点是让每个项目有清楚的背景、过程和结果。       |
| 8. 服务 / 工作室官网  | 咨询、设计服务、开发外包、营销服务         | 页面目标是让潜在客户理解服务并咨询。结构建议：服务定位、服务内容、合作流程、案例、客户评价、报价方式、预约咨询。文案要强调可信度和交付能力。       |
| 9. 活动 / 课程招募页  | Workshop、训练营、直播、线下活动、课程报名 | 页面目标是报名转化。结构建议：活动主题、适合人群、时间地点、讲师介绍、课程大纲、收获、价格、报名按钮、FAQ。重点是降低报名犹豫。            |
| 10. 数据报告 / 洞察页 | 行业报告、趋势分析、调研结果、年度总结       | 页面要有信息密度和专业感。结构建议：报告标题、核心结论、关键数据、图表模块、分章节分析、方法说明、下载/分享 CTA。适合图表、数字卡片、引用块。    |

## 10 个风格方向

| 风格              | 视觉特征                  | agent 执行指导                                                        |
| --------------- | --------------------- | ----------------------------------------------------------------- |
| 1. 极简 SaaS 风    | 大留白、浅背景、清晰卡片、现代字体     | 使用白色或浅灰背景，主色控制在 1 个，按钮清晰，圆角中等，阴影轻。适合产品介绍页、等候名单页、功能发布页。            |
| 2. Bento Grid 风 | 模块化卡片、信息块清晰、网格布局      | 用不同大小的卡片组织内容，重点模块占大卡片，次要内容放小卡片。适合功能展示、产品卖点、个人主页、数据总结。             |
| 3. 新粗野派风        | 粗边框、硬阴影、高对比、大色块       | 使用黑色粗线框、强烈背景色、明显按钮、夸张标题。适合年轻化产品、创作者工具、社区项目。注意整体要有秩序，避免随机堆砌。       |
| 4. 新拟态风         | 柔和凸起、浅色背景、低对比阴影       | 使用浅灰、米白、淡蓝等背景，组件像从背景里浮起。适合轻量工具、健康类、个人页。注意对比度要足够，按钮和文字不能太淡。        |
| 5. 玻璃拟态风        | 半透明卡片、背景模糊、渐变光晕       | 使用深色或渐变背景，卡片加透明度、模糊、细边框。适合 AI、Web3、未来感产品。注意不要让背景干扰文字阅读。           |
| 6. 扁平插画风        | 简洁色块、插画图形、低复杂度 UI     | 使用明亮但克制的配色，图标和插画统一风格，卡片边界清楚。适合教育、知识讲解、活动报名、轻产品。                   |
| 7. 像素复古风        | 8-bit、低分辨率图标、等宽字体、游戏感 | 使用像素边框、硬切色块、复古按钮、终端感模块。适合游戏、开发者项目、meme 社区。文字排版要保持清楚，不要牺牲可读性。      |
| 8. 素雅 Muji 风    | 米白、低饱和、细线、克制排版        | 使用米色、浅灰、木色、淡黑文字。圆角小，装饰少，排版安静。适合个人主页、生活方式、知识内容、手作品牌。               |
| 9. 杂志编辑风        | 大标题、强排版、图文节奏、栏目感      | 使用大字号标题、清晰章节、图片穿插、引用块。适合知识讲解页、报告页、个人品牌页。重点是做出阅读节奏，不要全部堆成卡片。       |
| 10. 未来科技风       | 深色背景、渐变、光晕、线框、动态感     | 使用黑色/深蓝背景，搭配蓝紫绿等荧光色，加入网格、光线、数据面板。适合 AI、开发者工具、数据产品。动效可以更明显，但要控制数量。 |

## 给 agent 的组合规则

针对这 10 个方向和 10 个风格做成 **10 × 10 的组合矩阵**。比如「产品介绍页 + 极简 SaaS 风」「知识讲解页 + 杂志编辑风」「个人主页 + 素雅 Muji 风」「数据报告页 + 未来科技风」。这样 agent 不需要从零设计，只需要在明确边界内组合和微调。
